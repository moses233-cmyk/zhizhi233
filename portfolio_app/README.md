# Portfolio App 框架

本目录提供一个基于静态页面的个人资料与作品集展示程序框架，可用于快速启动展示型站点开发。

## 功能概览

- **展示模块**：关于我、作品集、客户反馈、联系方式等板块布局完善，适配桌面与移动端。
- **交互体验**：支持主题切换、作品分类筛选、平滑滚动、可视区域动态展示。
- **表单校验**：内置基础的前端表单校验逻辑，示例化提交提示。
- **易于扩展**：结构化的 HTML、CSS、JavaScript 文件，方便按需拓展功能或接入后端服务。

## 使用方式

1. 在本地启动静态服务器（例如 `npx serve` 或 IDE 内置静态服务），指向本目录。
2. 访问 `http://localhost:3000`（具体端口视工具而定），即可体验页面。
3. 根据实际需求调整 `index.html` 内的内容模块、`styles.css` 的样式以及 `app.js` 的交互逻辑。
